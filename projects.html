WaltZ (Bipedal walking robot)

March 2024 - Current

This is the main project undertaken at the Khoury Robotics Club. The purpose of the porject is to develop a low cost walking bipedal robot. The robot has 6 DoF. It is controlled by a RP2040 micro controller with an integrated 9 axis IMU. This project is in development and is expected to be used as an education tool in the coming undergraduate classes.

Spatially Handeling Arm Projectile EvadeR (SHAPER)

Nov 2023 - Dec 2023

Genetic Algortihm used to train a simulated robot arm to evade flying objects. This entire project was written using numpy and pygame without using any exsisting libraries. The algorith used a new multistep scoring function that incrementally increases the complexity of the target function needed to be learnt by the robot arms. This appraoch significantly imporved training time compared to traditional genetic algorithms.
